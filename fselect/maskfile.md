## fselect-name

```bash
cd "$(git root)"
echo -e "path\tsize\tmime\tline_count"
fselect "concat('[', path, '](../', path, ')'), size, mime, line_count \
    from . gitignore \
    where name like '%fselect%' \
    order by path"
```

## fselect-contains

```bash
cd "$(git root)"

    from . gitignore \
```

## fselect-Home-Today

> **Very slow!**

```bash
cd ~
echo -e "path\tsize"
RUST_BACKTRACE=1 fselect path, size where modified = today order by path
```
