The duckscript command line executable.

USAGE:
    duck [FLAGS] [FILE]

FLAGS:
    -h, --help      Prints help information
    -e, --eval      Evaluate script
    -l, --lint      Lints the provided file

ARGS:
    <FILE>          The script file to execute/lint
